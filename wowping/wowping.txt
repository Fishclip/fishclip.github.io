This tool builds a dynamic list of World of Warcraft realms and calculates
each realm's latency.  The in-game latency display will be greater than the
latency listed below, but the list will give an indication of the relative
latency of realms from Perth, Western Australia.

Realms with a latency of "Timeout" indicate the server is either not
contactable or it's latency is greater than 2 seconds.  A latency of
"Unknown" means the IP address of the server is not available so it's
latency cannot be calculated.

The IP address of a realm may change at any time and some realms change
more requently than others.  Once a realm's IP address is no longer is use
it may then be taken by a different realm.  For these reasons the older the
realm IP list is, the less likely it is to be accurate.  It isn't difficult
to obtain a current listing of realm IP addresses although one must use the
World of Warcraft client.  As I no longer have this installed on my PC,
this task is left to someone else.
